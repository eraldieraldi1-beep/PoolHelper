# PoolHelper

## Overview
PoolHelper is an overlay application designed to assist users in managing and optimizing their pool maintenance tasks. The application provides a user-friendly interface that allows users to track schedules, manage equipment, and monitor pool conditions.

## Features
- **Task Management**: Keep track of routine pool maintenance tasks and schedules.
- **Equipment Tracking**: Manage and log equipment usage and maintenance history.
- **Water Quality Monitoring**: Input data regarding water testing results and receive recommendations.
- **Notifications**: Set reminders for upcoming tasks and alerts for water quality issues.

## Installation
To install the PoolHelper application, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/eraldieraldi1-beep/PoolHelper.git
   ```
2. Navigate to the PoolHelper directory:
   ```bash
   cd PoolHelper
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
Once the application is running, you can:
- Create and manage your pool maintenance tasks.
- Log the equipment used during each maintenance session.
- Record test results and receive recommendations for pool care.
